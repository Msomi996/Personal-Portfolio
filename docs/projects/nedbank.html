Nedbank’s branding underwent a few changes and all Nedbank existing products had to be upgraded to match the new user interface requirements. We started by upgrading all the UI (User Interface), conducted user testing, and identified opportunities for enhancements. The relevant enhancements were presented, later designed, and tested before being built and deployed.
